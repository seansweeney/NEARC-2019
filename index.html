Migrating AGOL User Accounts Using the ArcGIS API for Python

Sean Sweeney, GIS Senior Programmer Analyst

City of Cambridge, MA

NEARC - Newry, ME

October 21, 2019

Cambridge is moving the majority of our ArcGIS Online users from AGOL logins to enterprise logins. Our goal is to move users in a controlled manner - not only their content, but their profile and other aspects of their AGOL identity - from their old AGOL login to their new enterprise account. I'll demonstrate some of the components of the ArcGIS API for Python we're using to accomplish this, point out a few curveballs we've run into along the way, and also show final Python script that (mostly) automates the process.
